Create `backend/scripts/test_ai_provider.py` — a standalone script that verifies the AI abstraction works with real credentials. The script:

1. Imports `call_ai` from `app.core.ai_service` and `settings` from `app.core.config`
2. Prints current AI_PROVIDER and AI_MODEL (NOT the API key)
3. Sends a simple test message: `[{"role": "user", "content": "Reply with exactly: HELLO"}]`
4. Prints whether the response contains "HELLO"
5. Sends a plan-chat-style message with a small JSON context and instruction to return a JSON patch
6. Prints whether the response is valid JSON

Usage instructions printed at top:
```

# 1. Set AI_PROVIDER and AI_API_KEY in your .env
# 2. cd backend && python -m scripts.test_ai_provider
```
