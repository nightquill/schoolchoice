**BUG-01 — Rolling 24h rate limit** (per D-discretion, per RESEARCH.md BUG-01 code example):

In `backend/app/services/plan_chat_service.py`, replace the `_check_and_increment_rate_limit` function. The current implementation uses `date.today().isoformat()` as part of the key, which resets at midnight. Replace with a rolling 24-hour window:

1. Remove the `_rate_limit_key` helper that uses `date.today()`.
2. Replace `_check_and_increment_rate_limit` with this logic:
   - Key format: `f"{counsellor_id}:{plan.id}"` (no date prefix)
   - Read `counts = dict(plan.chat_request_counts or {})`
   - Get entry: `entry = counts.get(key, {"count": 0, "window_start": None})`
   - Parse `window_start` as ISO datetime. If `now - window_start > timedelta(hours=24)`, reset entry to `{"count": 0, "window_start": now.isoformat()}`
   - If no `window_start`, set it to `now.isoformat()`
   - If `entry["count"] >= 20`, raise HTTPException 429
   - Increment `entry["count"]`, write back to `plan.chat_request_counts`
   - Import: `from datetime import datetime, timezone, timedelta`

**BUG-02 — Data completeness indicator** (per RESEARCH.md BUG-02 code example):

In `backend/app/services/matchmaker_v2.py`:
1. Add `data_completeness: float` field to the `MatchResult` dataclass, after the `rationale` field. Type: `float` (0.0-1.0).
2. Add a `compute_data_completeness(student_data: dict) -> float` function:
   - `grades = student_data.get("grades_by_code", {})`
   - Count compulsory subjects present: `sum(1 for c in ["CHLA", "ENGL", "MATH", "CSD"] if c in grades and grades[c])`
   - `compulsory_score = compulsory_present / 4`
   - Count non-compulsory grades present, elective_score = `min(len(present) / 2, 1.0)`

3. In `run_matching()` (or wherever MatchResult is constructed), call `compute_data_completeness(student_data)` and pass the result as the `data_completeness` argument.

**SQLite dialect guard** (per RESEARCH.md Pitfall 6, Open Question 1 — option (a)):

In `backend/app/main.py`, wrap the entire "Runtime column migrations" block (lines ~138-160) in a dialect check:

```python
# Runtime column migrations (PostgreSQL only — SQLite does not support IF NOT EXISTS in ALTER TABLE)
if engine.dialect.name == "postgresql":
    from sqlalchemy import text as _sql_text
    with engine.connect() as _conn:
        _conn.execute(_sql_text("ALTER TABLE students ADD COLUMN IF NOT EXISTS personal_statement TEXT"))
        _conn.execute(_sql_text("ALTER TABLE student_school_targets ADD COLUMN IF NOT EXISTS intended_majors JSON"))
        _conn.execute(_sql_text("ALTER TABLE student_school_targets ADD COLUMN IF NOT EXISTS year_of_entry INTEGER"))
        _conn.execute(_sql_text("ALTER TABLE schools ADD COLUMN IF NOT EXISTS is_custom BOOLEAN DEFAULT FALSE"))
        _conn.execute(_sql_text("ALTER TABLE schools ADD COLUMN IF NOT EXISTS major_requirements JSON"))
        _conn.execute(_sql_text("ALTER TABLE schools DROP CONSTRAINT IF EXISTS ck_schools_type"))
        _conn.execute(_sql_text(
            "ALTER TABLE schools ADD CONSTRAINT ck_schools_type CHECK "
            "(type IN ('UNIVERSITY', 'POLYTECHNIC', 'COMMUNITY_COLLEGE', 'VOCATIONAL', 'HIGH_SCHOOL') OR type IS NULL)"
        ))
        _conn.execute(_sql_text("ALTER TABLE students ADD COLUMN IF NOT EXISTS is_graduated BOOLEAN DEFAULT FALSE"))
        _conn.execute(_sql_text("ALTER TABLE students ADD COLUMN IF NOT EXISTS graduation_year INTEGER"))
        _conn.execute(_sql_text("ALTER TABLE students ADD COLUMN IF NOT EXISTS final_school_id UUID REFERENCES schools(id) ON DELETE SET NULL"))
        _conn.execute(_sql_text("ALTER TABLE students ADD COLUMN IF NOT EXISTS final_major VARCHAR(255)"))
        _conn.execute(_sql_text("ALTER TABLE academic_plans ADD COLUMN IF NOT EXISTS template_id VARCHAR(50) DEFAULT 'professional'"))
        _conn.execute(_sql_text("ALTER TABLE academic_plans ADD COLUMN IF NOT EXISTS overrides JSON DEFAULT '{}'"))
        _conn.execute(_sql_text("ALTER TABLE academic_plans ADD COLUMN IF NOT EXISTS chat_request_counts JSON DEFAULT '{}'"))
        _conn.commit()
```

Also move the `from sqlalchemy import text as _sql_text` import inside the `if` block so it doesn't execute at module level when running tests.
