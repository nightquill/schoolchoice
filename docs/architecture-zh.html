產品概述

SchoolChoice 為學校升學輔導員提供一站式工作平台，整合學生學業數據管理、以真實 JUPAS 入學計分公式評估大學匹配度，以及生成個人化升學計劃書——取代試算表、主觀估算和人手交叉比對，以數據驅動的工作流程提升輔導效率。

核心功能

學生數據管理

完整的學業檔案：支援多次應考（模擬試、校內試、正式考試）的 DSE 成績、IELTS 分數、教師評語、課外活動及獎項紀錄。支援 CSV/Excel 批量匯入，方便全班資料輸入。

大學匹配分析

依據每個 JUPAS 課程公佈的計分公式計算入學概率。每個課程有獨立的科目比重、最佳 N 科選取規則及評分量表。結果以實際公佈的入學統計數據（中位數、下四分位數）作校準。

升學計劃書生成

生成結合量化匹配數據與 AI 撰寫建議的個人化 HTML 計劃書。三種範本風格可選。支援逐段編輯、AI 對話微調，以及列印輸出。

群組分析

班級成績分佈、目標院校熱門度、高風險學生識別。可按考試類別篩選，追蹤從模擬試到正式考試的進步情況。批量成績輸入格可高效處理數據。

列印及輔導

當輔導員為學生選擇目標大學時，系統會即時運行匹配引擎。每個目標院校會顯示色彩編碼的匹配百分比、資格狀態（合格/不合格及具體未達標項目），以及當學生預測分數低於該課程已取錄學生的下四分位數時，會顯示「高風險」警示。

匹配引擎使用各大學就各課程公佈的專屬計分公式——而非通用演算法。同一學生報讀港大工程學院與中大商學院，會因為兩個課程的科目比重不同而得出不同的分數。

多年成績追蹤

學生通常在中五應考模擬試，中六應考正式 DSE。系統追蹤多次應考及多年的成績，以最新數據計算預測成績，並以最佳可用分數進行大學匹配。輔導員可查看成績趨勢，據此調整建議。

系統設計

層次技術備註
前端React, Vite, TailwindCSS單頁應用。響應式設計。計劃書生成支援即時串流。
後端Python (FastAPI)105+ API 端點。JWT 認證。速率限制。
資料庫SQLite（開發）/ PostgreSQL（正式環境）12 個資料表。可攜式 UUID。JSONB 彈性欄位。
AI 整合LiteLLM（供應商無關）用於計劃書建議文字及對話。不用於評分——評分為確定性計算。
部署Vercel（前端）+ Railway（後端）標準雲端託管。不需 GPU 或特殊基礎設施。
關鍵設計決策：評分引擎完全採用確定性計算——無機器學習、無訓練數據、無黑盒模型。每個分數均可追溯至大學公佈的計分公式及已核實的入學統計數據。AI 僅用於生成升學計劃書中的建議文字，不影響量化推薦的準確性。

多租戶架構

每間學校在獨立的數據空間中運作。A 校的輔導員無法查看 B 校的學生資料。數據隔離透過機構範圍的 JWT 令牌在 API 層強制執行——每個資料庫查詢均自動按輔導員所屬機構過濾。

私隱及合規

《個人資料（私隱）條例》合規

逐學生追蹤同意紀錄（資料處理、AI 分析、匯出、共享）。同意可授予及撤回，附完整審計軌跡。學生姓名在傳送至第三方 AI 供應商前會進行匿名化處理。

數據保護

個人識別資料（PII）封鎖清單防止敏感欄位（醫療記錄、紀律處分記錄、身份證號碼、出生日期）被納入 AI 提示。所有 AI 輸出均經注入攻擊消毒。密碼以 bcrypt 雜湊處理。JWT 令牌 60 分鐘後過期。
